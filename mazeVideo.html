About

This video, made before the RCAP competition and submitted for team qualification, showcases our 1st place-winning robot, which also earned Best Hardware Design. Powered by advanced software, the robot uses graph search algorithms for path planning, OpenMV with custom Python code for visual victim detection, PID controllers for precise navigation, and an innovative rescue kit dropper. After this video, the robot was further upgraded in both hardware and software.
